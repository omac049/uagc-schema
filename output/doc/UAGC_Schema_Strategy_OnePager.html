What We Have Today

Foundation
Organization, WebSite, WebPage -- the identity layer for UAGC across every page.
View JSON-LD example 
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "University of Arizona Global Campus",
  "url": "https://www.uagc.edu",
  "logo": "https://www.uagc.edu/logo.png",
  "sameAs": [
    "https://www.facebook.com/UABORAGC",
    "https://www.linkedin.com/school/uagc"
  ]
}
Navigation
BreadcrumbList signals that map the Home → Section → Page hierarchy.
View JSON-LD example 
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://www.uagc.edu"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Online Degrees",
      "item": "https://www.uagc.edu/online-degrees"
    }
  ]
}
Content
Article, BlogPosting, ImageObject & VideoObject on editorial templates.
View JSON-LD example 
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "5 Tips for Online Learning Success",
  "author": {
    "@type": "Organization",
    "name": "UAGC"
  },
  "datePublished": "2025-09-15",
  "image": "https://www.uagc.edu/blog/hero.jpg",
  "publisher": {
    "@type": "Organization",
    "name": "University of Arizona Global Campus"
  }
}
Q&A
FAQPage markup on sections that genuinely contain FAQ content.
View JSON-LD example 
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "How do I apply to UAGC?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "You can apply online at uagc.edu/apply..."
      }
    }
  ]
}
Academic
Course and Program structured data on program/course templates.

People
Person schema on professor and faculty profile pages.
View JSON-LD example 
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "Dr. Jane Smith",
  "jobTitle": "Associate Professor of Psychology",
  "worksFor": {
    "@type": "Organization",
    "name": "University of Arizona Global Campus"
  },
  "alumniOf": "University of Arizona",
  "image": "https://www.uagc.edu/faculty/jsmith.jpg"
}
Discovery
ItemList and SearchAction for internal search and category pages.
View JSON-LD example 
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "url": "https://www.uagc.edu",
  "potentialAction": {
    "@type": "SearchAction",
    "target": {
      "@type": "EntryPoint",
      "urlTemplate": "https://www.uagc.edu/search?q={query}"
    },
    "query-input": "required name=query"
  }
}
